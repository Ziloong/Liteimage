## 1. Introduction

**LiteImage** is a native macOS image compression tool with two compression engines:

| Engine | Description | Advantage |
|--------|-------------|-----------|
| **TinyPNG Cloud** | Uses TinyPNG API | Higher compression, PNG 24-bit transparency support |
| **Local Engine** | Uses pngquant / gifsicle | Fully offline, no network required |

Also supports **Video to GIF** conversion, powered by gifski + ffmpeg for high-quality output.

## 2. Usage

### System Requirements

- macOS 12.0 or later
- Optional: TinyPNG API Key ([Free申请](https://tinypng.com/developers), 500 credits/month)

### Installation

1. Download the latest Release
2. Drag `LiteImage.app` to Applications
3. On first launch, allow the app in **System Settings → Privacy & Security**

## 3. Features

### Image Compression

- PNG / JPG / GIF support
- TinyPNG cloud + local engine dual mode
- **Local engine quality presets**: Ultra Low / Low / Medium / High (default), covering quality range 10-95
- **Resize by long edge**: Custom pixel input with presets (1920 / 1280 / 800)
- Real-time stats (files compressed, space saved, average ratio)
- Overwrite original or save as copy
- Compression log

### Video to GIF

- High-quality encoding via gifski
- Lightweight ffmpeg for frame extraction (app bundle only 4.6MB)
- Quality presets: Low / Medium / High
- Adjustable width and frame rate
- Real-time conversion log

### Other

- **Multi-language**: Auto-detects system language (简体中文 / English)
- **Native experience**: Built with SwiftUI, smooth and responsive
- **Secure**: All compression done locally (except TinyPNG mode which requires network)
